# Crop-Recommendation-System

An AI-powered system designed for precision agriculture. This project uses a **Random Forest Classifier** to recommend the most sustainable crop based on soil nutrients (NPK) and climatic data.

## Features
* **Machine Learning:** Random Forest Model with 99.1% accuracy.
* **Sustainability:** Optimized for responsible fertilizer use and soil health.
* **Tech Stack:** Python, Scikit-Learn, Pandas, Matplotlib.

## Dataset
Includes soil NPK levels, temperature, humidity, pH, and rainfall.
